# RULE-STACK-001

## Rule

[NEEDS CLARIFICATION: 记录本仓库唯一或优先使用的包管理器、运行环境和命令前缀。]

## Details

只有当仓库事实能确认包管理器、运行环境或命令前缀时，才启用这条规则。例如 `uv run ...`、`npm run ...`、`pnpm ...`、`bundle exec ...` 或项目自定义 wrapper。不要从模板或其他仓库继承命令。

## Evidence

- [NEEDS CLARIFICATION: manifest、lockfile、tool config、README、Makefile/justfile 或 CI]

## Guard

- [NEEDS CLARIFICATION: install check、verification skill、runner、CI 或未配置]
